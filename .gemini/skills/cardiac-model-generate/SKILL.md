# Cardiac Model Generate Skill

## When to use this skill

Use this skill when a user needs codes/result from a cardiac model

## How to Use this skill

1. Based on the model name provided, choose and copy the corresponding model from reference.

2. Change parameters if provided parameters are different from the ones in file copied.

### Returns

Html simulation file copied.
